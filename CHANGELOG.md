## [v3.3.0] - 2026-02-02

### Highlights

- New Cobra CLI for interacting with Aha! API from the command line
- Comprehensive idea-feature-release reporting with multiple output formats

### Added

- CLI: Cobra CLI with commands for ideas, features, products, releases, and reports ([`28c5f26`](https://github.com/grokify/go-aha/commit/28c5f26))
- CLI: `--inflate` flag to fetch full idea details including categories ([`30083d1`](https://github.com/grokify/go-aha/commit/30083d1))
- CLI: `--sort` flag for votes/created/updated ordering ([`30083d1`](https://github.com/grokify/go-aha/commit/30083d1))
- CLI: `--fail-on-error` flag for strict feature fetch error handling ([`30083d1`](https://github.com/grokify/go-aha/commit/30083d1))
- CLI: Progress bar during idea inflation ([`30083d1`](https://github.com/grokify/go-aha/commit/30083d1))
- SDK: `IdeaFeatureReport` and `IdeaFeatureReportSet` for comprehensive idea-feature-release reporting ([`e33b815`](https://github.com/grokify/go-aha/commit/e33b815))
- SDK: `GetIdeaFeatureReports()` with automatic pagination via `FetchAll` option ([`e33b815`](https://github.com/grokify/go-aha/commit/e33b815))
- SDK: `ListIdeas()` and `ListIdeasExecute()` in `IdeasAPIService` ([`cbd3b00`](https://github.com/grokify/go-aha/commit/cbd3b00))

### Fixed

- `CustomField.Value` now supports any type (string, array, etc.) matching actual Aha! API responses ([`f895ca6`](https://github.com/grokify/go-aha/commit/f895ca6))
- `Idea` model fields (`categories`, `votes`, timestamps) are now optional to handle varying API responses ([`961b1ca`](https://github.com/grokify/go-aha/commit/961b1ca))

### Dependencies

- Bump `google.golang.org/api` from 0.250.0 to 0.256.0 ([`b7f5f06`](https://github.com/grokify/go-aha/commit/b7f5f06))
- Bump `github.com/grokify/mogo` from 0.70.23 to 0.71.7 ([`bbf6706`](https://github.com/grokify/go-aha/commit/bbf6706))
- Bump `actions/checkout` from 5 to 6 ([`135a81b`](https://github.com/grokify/go-aha/commit/135a81b))

### Documentation

- MkDocs documentation site with Getting Started, CLI Reference, SDK Guide, and Examples ([`503c07a`](https://github.com/grokify/go-aha/commit/503c07a))
- README updated with quick start, API reference table, and corrected examples ([`7ae412a`](https://github.com/grokify/go-aha/commit/7ae412a))
